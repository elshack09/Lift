# Lift
MERN Stack app that teaches beginners how to lift weights.

## Heroku Deployed App
https://glacial-woodland-20110.herokuapp.com/

## Technologies Used
### Front-End
* Reactstrap
* Axios
* React-Router-Dom
### Back-End
* MongoDB
* Mongoose
* Express
* Node.js
### Design
* Sketch
* Craft
